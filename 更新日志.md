# 更新日志

## 2026-05-03

- 版本号升级到 2.0.2
- 修复 Release 安装版默认窗口过小的问题，默认窗口调整为 1280x800，最小尺寸调整为 960x640，并允许用户自由缩放窗口

- 版本号升级到 2.0.1

- 批量导入自适应多种JSON格式（数组/包装对象/扁平导出/简化凭证）

- 放大账号详情和可用模型窗口，优化模型列表多列展示和详情信息布局
- 批量导入、批量检查、批量刷新新增底部悬浮进度提示
- 修复批量健康检查异常计数与列表状态不同步的问题，并补充“已封禁”状态筛选
